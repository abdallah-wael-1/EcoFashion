# Sustainable Fashion & Upcycling Marketplace

A modern React frontend for a web application focused on sustainable fashion, upcycling, and swapping second-hand clothing.

## Features

- Item listing with condition tags
- Swap request logic for trading items
- User profiles and trust scores
- Style feed based on user preferences
- Dashboard for sellers
- Responsive design

## Tech Stack

- React with Vite
- React Router for routing
- Context API for global state
- Modern CSS for styling

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:5174](http://localhost:5174) in your browser.
